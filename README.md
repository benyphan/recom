# Recom — Система рекомендаций товаров

 Современная платформа электронной коммерции с персонализированными рекомендациями на основе действий пользователя.

## Возможности

- **Персональные рекомендации** — умный алгоритм анализирует просмотры, лайки и покупки пользователя и предлагает релевантные товары
- **Content-Based рекомендации** — TF-IDF векторизация описаний товаров для поиска похожих позиций
- **История действий** — отслеживание просмотров, избранного и покупок
- **Система отзывов** — пользователи могут оставлять отзывы с оценкой
- **Админ-панель** — управление товарами и пользователями
- **Адаптивный дизайн** — красивый UI с анимациями

## Технологии

- **Backend**: Flask 3.x
- **Database**: SQLite (SQLAlchemy)
- **Auth**: Flask-Login
- **Frontend**: Jinja2 + Tailwind CSS + Animate.css

## Установка

```bash
pip install -r requirements.txt
python run.py
```

Откройте http://127.0.0.1:5000


## API Endpoints

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/register` | POST | Регистрация |
| `/api/login` | POST | Вход |
| `/api/logout` | POST | Выход |
| `/api/products/<id>/action` | POST | Действие (view/like/buy) |
| `/api/products/<id>/reviews` | POST | Создать отзыв |
| `/api/recommendations` | GET | Рекомендации |
| `/admin` | GET | Админ-панель |
| `/admin/upload` | POST | Загрузка изображений |
